colors-picker

（一个类似浏览器调试工具中颜色选择器的H5版本的小组件）

引入

首先你需要通过命令行安装，就像这样： 

npm install --save colors-picker

安装好了以后，引入即可： 

import ColorsPicker from 'colors-picker';

当然，你也可以通过CDN的方式引入： 

<script src='https://cdn.jsdelivr.net/npm/colors-picker@0'></script>

语法 

ColorsPicker(target, initColor).then(function (color) {
    // todo
});

其中，target表示页面交互DOM，initColor是初始化颜色（合法的css值即可，格式无要求）。

最后，通过then传递一个回调，其中的形参color表示你选择的颜色值。

例子

让我们举个例子：右边这个色块就是运行的效果： （你可以点击一下试试效果，当然，色块的UI可以自由定义）。

要实现那样的色块，首先，你需要在页面中准备一个可点击的节点，比如一个span标签： 

<span id='target'></span>

然后，就很简单了： 

ColorsPicker(document.getElementById('target'), '#1eff0087').then(function (color) {
    document.getElementById('target1').style.backgroundColor = color;
});
